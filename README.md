# Tonika — Music Theory in Real Time

Tonika is a lightweight open-source web app for learning music theory by playing.  
Visualize chords, scales, and harmonic relationships instantly on piano or guitar—without heavy setup or subscription costs.

---

##  Features
- **Real-time MIDI input** — connect a keyboard and see theory under your fingers.
- **Dual display** — piano and guitar fretboard views; toggle as needed.
- **Smart chord recognition** — supports triads, sevenths, extensions, inversions, slash chords.
- **Scale highlighting** — view major, minor, modal, pentatonic, chromatic scales; root and scale tones color-coded.
- **Big chord banner** — large, clear chord name helps with practice and composition.
- **Editable theory data** — scales and chords managed via JSON files for easy customization.
- **Dependency-free** — vanilla JS, runs in browser; no install needed for tech-savvy or learners alike.

---

##  Purpose
Tonika is not a DAW or notation editor—it’s a **practice and composition companion** that:
- Demonstrates chord/scale relationships dynamically
- Helps embed theoretical concepts through real-time interaction
- Acts as a **scratchpad for ideas** and early compositional sketches
- Provides visual reinforcement for training your music intuition
