Drop‑in Components

Tonika lets you drop *onika modules into cards with a single attribute: 

<div class="tk-card" data-onika="Clavonika" data-target="#clavonika-slot"></div>

No frameworks, no build steps required for users — just include the module’s script and Tonika mounts it.

Components

  • Clavonika — Piano keyboard UI
  • Jackonika — MIDI‑in stompbox / monitor
  • Backplanika — Micro router & patching
  • Midonika — MIDI visualizer

Each module exposes a global with a mount(el, opts) method.

Docs (Draft)

  1. Copy your component bundle into /public/components/.
  2. Add <script defer src="/components/your.onika.iife.js"></script>.
  3. Add a card with data-onika="YourGlobalName" and data-target to an element inside the card.

That’s it. No config files needed.
